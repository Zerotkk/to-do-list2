# 项目任务看板

这是一个基于 React + Vite 的多项目 To Do List。

## 正确启动方式

不要直接双击 `index.html`，React/Vite 需要开发服务器编译后才能运行。

方式一：双击运行：

```powershell
启动项目.bat
```

方式二：手动运行：

```powershell
cd "C:\Users\46183\Desktop\to do list"
npm.cmd install
npm.cmd run dev
```

启动成功后，终端会显示类似：

```text
Local: http://localhost:5173/
```

把这个地址复制到浏览器打开即可。
